<Dokerfile 생성>
1. 이미지를 만들 프로젝트에 루트 디렉토리에 Dokerfile을 생성한다.

    - Dokerfile 최상단에는 무조건 from base Image를 써준다.


    - 예시(
        # 1. OpenJDK 21 이미지를 베이스 이미지로 사용
        FROM eclipse-temurin:21-jdk

        # 2. JAR 파일을 컨테이너 내부로 복사
        ARG JAR_FILE=build/libs/*.jar
        COPY ${JAR_FILE} app.jar

        # 3. 포트 오픈 (Spring Boot 기본 포트 8080)
        EXPOSE 8080

        # 4. 애플리케이션 실행 명령
        ENTRYPOINT ["java", "-jar", "app.jar"]
    )


    - JAR 빌드하기
        자바를 이용한 프로젝트 경우 jar파일 빌드.


    - Docker 이미지 빌드 <- 이 단계에서 이미지 생성.
        docker build -t my-spring-app .

    
    - Docker 컨테이너 실행
        docker run -p 8080:8080 my-spring-app
